# react-native-pgw-2c2p

React Native 2c2p Payment Gateway SDK

## Installation

```sh
npm install react-native-pgw-2c2p
```

## Usage

```js
import { multiply } from 'react-native-pgw-2c2p';

// ...

const result = await multiply(3, 7);
```
